## Run Locally

Clone the project

```bash
  git clone [https://link-to-project](https://github.com/yDiony/WalletApp.git)
```

Go to the project directory

```bash
  cd WalletApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
